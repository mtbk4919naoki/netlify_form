## 手順

1. GitHubリポジトリを作成します

2. NetlifyとGitHubを連携します

3. リポジトリを選択します

4. デプロイするブランチ、公開するディレクトリを設定します

5. netlify formを利用するフォームにnetlify属性を追加します。

6. GitHubリポジトリにプッシュします
